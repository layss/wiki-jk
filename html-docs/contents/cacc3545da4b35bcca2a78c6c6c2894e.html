Editar dados Professor

UMLUseCase
Especificação de Requisitos Funcionais :: Casos de Uso :: Editar dados Professor

Description

Prioridade:Essencial

Ator (es): Administrador

Requisitos associados: Não há

Descrição: Este caso de caso de uso permitir editar os dados dos professores.

Pré-condições: Professor deverá está cadastrado no sistema.·.

Pós-condições: O administrador terá como editar os dados dos professores no sistema.

Fluxo principal:
1. O caso de uso inicia quando o administrador seleciona a opção “Editar Professor”.

  1. O sistema apresenta a mensagem: “TEM CERTEZA QUE DESEJA EDITAR” [FA002].

  2. O administrador seleciona a opção “SIM”.

  3. O sistema apresenta a mensagem: ALTERADO COM SUCESSO!

  4. O caso de uso de encerra.

Fluxos alternativos

[FA002] – Editar Professor

  1. No passo (2) do fluxo principal caso o administrador seleciona a opção “NÃO”

  2. O caso de uso se encerra
