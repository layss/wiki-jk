Remover dados Professor

UMLUseCase
Especificação de Requisitos Funcionais :: Casos de Uso :: Remover dados Professor

Description

Prioridade: Essencial

Ator (es): Administrador

Requisitos associados: Não há

Descrição: Este caso de caso de uso permitir remover o professor

Pré-condições: Professor deverá está cadastrado no sistema.

Pós-condições: O administrador terá como remover o professor do sistema.

Fluxo principal

  1. O caso de uso inicia quando o administrador seleciona a opção “Remover Professor”.

  2. O sistema apresenta a mensagem: “TEM CERTEZA QUE DESEJA REMOVER” [FA003]

  3. O administrador seleciona a opção “SIM”.

  4. O sistema apresenta a mensagem: REMOVIDO COM SUCESSO!

  5. O caso de uso de encerra.

Fluxos alternativos

[FA004] – Remover Professor

  1. No passo (2) do fluxo principal caso o administrador seleciona a opção “NÃO”

  2. O caso de uso se encerra
